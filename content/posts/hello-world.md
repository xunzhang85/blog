## 终于动手了

想写博客这件事已经拖了好几年。今天终于把这个念头变成了现实。

选择 **Hugo + Stack** 的理由很简单：

- ⚡ **快** — Hugo 的构建速度是毫秒级的，写完文章刷新即可见
- 🎨 **好看** — Stack 主题暗色模式深得我心
- 🛠 **好维护** — 纯 Markdown 写作，Git 管理，零运维

<!--more-->

## 技术栈

这个博客的技术栈：

| 工具 | 用途 |
|------|------|
| [Hugo](https://gohugo.io) | 静态站点生成器 |
| [Stack](https://github.com/CaiJimmy/hugo-theme-stack) | 主题 |
| GitHub Pages | 免费托管 |
| Markdown | 内容格式 |

## 接下来的计划

1. 记录日常开发中踩过的坑
2. 分享一些有意思的项目
3. 写读书笔记和技术思考

用起来再说，完美主义是行动力的敌人。
